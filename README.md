# Bitneedle Format

Authoritative public specification for Bitneedle picture records: playable images that carry recoverable record payload bytes in a visible spiral groove.

The current specification draft is:

- [Bitneedle Picture Record Format](spec/bitneedle-picture-record-format.md)

This repository is intended to hold the public format specification, segment registry, and conformance vectors. Application code lives in the implementation repositories.

## Status

Draft. The wire format is implemented by the Bitneedle reference player and creator tooling, but the public conformance vector set is still being prepared.

Existing internal golden files are useful implementation regression fixtures, but should not yet be treated as normative examples until regenerated against the current public header profile.
